Nemobot Setup Manual
================================

■ Requirements

- Minecraft Java Edition
- Node.js
- Python 3.12+ with uv (https://docs.astral.sh/uv/)
- NVIDIA GPU (24GB+ VRAM recommended)
- vLLM

■ Installation

  git clone https://github.com/soy-tuber/minecraft-ai-wrapper.git
  cd minecraft-ai-wrapper

  # Bot (Node.js)
  cd bot && npm install && cd ..

  # Brain (Python)
  cd brain && uv sync && cd ..

  # Minecraft Server
  cd server
  # Download server.jar (Minecraft 1.21.x) and place it here
  # Copy server.properties.example to server.properties and configure
  # Accept EULA: echo "eula=true" > eula.txt
  cd ..

  # vLLM (can be in a separate environment)
  uv add vllm

■ Starting (3 terminals)

  1. Launch Minecraft (Windows / Mac)

  2. Terminal 1: Start the server
     cd server && ./start.sh

  3. Connect to the server from Minecraft (localhost)

  4. Terminal 2: Start vLLM
     uv run vllm serve nvidia/NVIDIA-Nemotron-Nano-9B-v2-Japanese \
         --max-model-len 32768 --gpu-memory-utilization 0.9 --trust-remote-code

  5. Terminal 3: Start the wrapper
     ./start.sh

■ Stopping

  Wrapper: Ctrl+C or ./start.sh stop
  vLLM: Ctrl+C
  Server: Ctrl+C

■ Usage

  Talk to the bot via in-game chat.
  Works in both English and Japanese.

  Command examples:
    "come here"          → Bot follows you
    "cut a tree"         → Chop the nearest tree
    "kill the sheep"     → Attack one sheep
    "hunt all sheep"     → Hunt all nearby sheep
    "guard me"           → Bodyguard mode
    "collect items"      → Pick up dropped items
    "give me your stuff" → Deliver items to player
    "look around"        → Report nearby entities
    "dance!"             → Dance
    "stop"               → Stop all actions

■ Chat UI

  A web server starts on port 3001 when the bot launches.
  Access http://localhost:3001 to control the bot from a browser.

■ Troubleshooting

  Q: Bot keeps saying it doesn't understand
  A: Say "stop" to reset the conversation history.

  Q: Bot doesn't pick up items
  A: Say "collect items" to gather them, then "give me" to receive them.

  Q: How to give the bot an axe
  A: Drop an axe near the bot. Say "collect" and it will pick up and auto-equip.
